# Advent of Code 2025

https://adventofcode.com/2025/

## Setup

```bash
export AOC_SESSION="" # AOC session cookie
make install-hooks    # Install git hooks
make inputs           # Fetch inputs
```

## Run

```bash
make run DAY=01
```

to check answer,

```bash
make answer DAY=01 PART=1 ANSWER=1234
```
